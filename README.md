# Atlas picture Unpacker

## Requirements

- Python (with pip)
- [Pillow](https://python-pillow.org)

## Installation

```sh
$ sudo easy_install pip
$ pip install Pillow
```

## Usage

`test.atlas` and `test.png` will product a dir named test:
```sh
$ atlascutter.py test
```
or
```sh
$ atlascutter.py test.atlas
```
